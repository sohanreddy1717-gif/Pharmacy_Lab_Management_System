# Pharmacy Lab Management System

## Overview

The Pharmacy Lab Management System is a Database Management System (DBMS) project developed to efficiently manage pharmacy operations. The system maintains records of medicines, customers, suppliers, and transactions while ensuring organized storage and retrieval of information.

The project demonstrates the application of database design principles, SQL, and Python to build a simple and effective pharmacy management solution.

---

## Features

- Medicine inventory management
- Customer record management
- Supplier information management
- Purchase and sales tracking
- Database-driven operations
- SQL-based data storage and retrieval
- User-friendly implementation

---

## Technologies Used

- Python
- SQL
- SQLite / MySQL
- Database Management Systems (DBMS)

## Installation

1. Clone the repository.

```bash
git clone https://github.com/sohanreddy1717-gif/Pharmacy_Lab_Management_System.git
```

2. Navigate to the project directory.

```bash
cd Pharmacy_Lab_Management_System
```

3. Install the required dependencies.

```bash
pip install -r requirements.txt
```

4. Configure the database using the provided SQL file.

5. Run the application.
